Pet Framework

This framework will:

  • Fully customizable egg opening animations with unique effects for each rarity
  • Advanced progression systems including Hard Pity, Soft Pity and Global Pity support
  • Built-in pet index with collection tracking and undiscovered entries
  • Custom pet states and animations including Walking, Standing and Flying behaviors
  • Client-side movement powered by BulkMoveTo for smooth performance on all devices
  • Raycast-based positioning keeps pets naturally aligned with the terrain
  • Complete pet management systems including Equip, Equip Best, Mass Delete and Upgrades
  • Modular architecture designed for future expansions and custom mechanics
  • Optimized for large pet counts while maintaining smooth gameplay
  • Easy integration with custom currencies, inventories and data frameworks

Data Saving + Inventory Framework

This framework will:

  • Securely save and load player data using ProfileStore for maximum reliability
  • Automatically synchronize data with UI elements and leaderstats in real time
  • Built with Object-Oriented Programming for clean, scalable and maintainable code
  • Highly optimized architecture designed for performance and simplicity
  • Flexible data structures allow inventories to display items in any format, including stackable items and custom layouts
  • Modular design makes it easy to integrate currencies, upgrades and additional systems
  • Automatic session handling and safe data release prevent data corruption issues
  • Simple API allows developers to create, modify and access player data with minimal code
  • Supports custom inventory behaviors, categories and filtering systems
  • Designed to work seamlessly with existing frameworks and future game expansions

Combat and Abilities Framework

This framework will:

  • Use raycast-based hit detection for precise and reliable hit registration
  • Feature a fully modular architecture designed for easy expansion and customization
  • Automatically synchronize combat states and cooldowns with UI elements
  • Support melee attacks, abilities, combo chains and projectile systems
  • Include movement mechanics such as dashes and directional abilities
  • Advanced combat calculations support critical hits, backstabs and custom damage modifiers
  • Flexible skill design allows new weapons and abilities to be added with minimal effort
  • Built to handle complex combat interactions while maintaining smooth performance
  • Easy integration with custom stats, buffs, debuffs and progression systems
  • Designed to serve as a foundation for RPG, simulator and action-based experiences

AI Framework

This framework will:

  • Feature a dynamic distance-based finite state machine for intelligent decision making
  • Use stochastic reaction latency to create natural and human-like response times
  • Implement close-quarters micro-flanking and adaptive positioning behaviors
  • Encapsulate state tracking through Object-Oriented memory systems for clean and maintainable AI logic
  • Allow NPCs to react dynamically to changing combat situations and player actions
  • Support customizable states such as Idle, Patrol, Chase, Attack and Retreat
  • Enable advanced movement behaviors with contextual repositioning and target evaluation
  • Designed with a fully modular architecture for easy expansion and customization
  • Optimized to handle multiple AI entities efficiently without sacrificing performance
  • Provide a strong foundation for RPG, survival, shooter and action-oriented experiences

Cross Server Matchmaking System

This framework will:

  • Match players across multiple servers for seamless and scalable gameplay
  • Feature a fully customizable architecture supporting any game mode and team size
  • Automatically create and manage reserved servers for isolated matches
  • Built with modular components for easy expansion and maintenance
  • Support flexible queue configurations, matchmaking rules and priority systems
  • Designed to handle large player counts while maintaining smooth performance
  • Allow custom matchmaking requirements such as skill levels, ranks or party systems
  • Provide reliable player synchronization throughout the entire matchmaking process
  • Simple integration makes it easy to connect with existing combat, data and progression frameworks
  • Built to serve as a foundation for competitive, cooperative and session-based experiences
